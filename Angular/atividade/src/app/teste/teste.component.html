Um pouco sobre o Framework Angular:

O Angular CLI (Command Line Interface) é uma poderosa ferramenta de linha de comando fornecida pela equipe do Angular para facilitar o desenvolvimento, o gerenciamento e a construção de aplicativos Angular. Ele oferece uma variedade de comandos pré-definidos que permitem criar rapidamente novos projetos, gerar componentes, serviços, módulos, e muito mais.
